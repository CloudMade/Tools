vectors — Vector data access¶

This module covers CloudMade’s Vector Stream Server API.

Vector objects¶

class vectors.Vector(apikey, referrer)¶

apikey is your API key that you’ve registered at CloudMade.

referrer is the host that is tied to your API key.

bbox(query)¶
Perform request with a given query.
tile(styleid, size, zoom, xtile, ytile)¶

Fetch SVG tile from tile server.

styleid is the CloudMade style id.

size is the size of the tile. Can be 256 or 64.

zoom is the zoom level on which you’re trying to fetch tile.

xtile and ytile are tile numbers.

Read more about these here.

Query objects¶

class vectors.Query(bbox)¶

Create new query builder. bbox is a sequnce of 4 coordinates representing southern_latitude, western_longitude, northern_latitude and eastern_longitude respectively.

types(*values)¶
Select which types of data should be returned. Can be any combination of POLYGON, LINE or POINT.
viewport(width, height)¶
The size of the vector data returned in pixels.
style(id)¶
Defines CloudMade style id which should be used for filtering and styling of data
zoom(level)¶
coords(type)¶
precision(level)¶
unused(action)¶
exclude(*bboxes)¶

Examples¶

Simple vector request:

from cloudmade import vectors

vector = vectors.Vector(apikey='Your API key', referer='your-domain.com')
query = vectors.Query([51.45, -0.1, 51.52, 0.0]).style('1').types(vectors.LINE, vectors.POLYGON)
data = vector.bbox(query)
with open('London.svg', 'w') as f:
    f.write(data)

Downloading part of San Francisco in one tile:

from cloudmade import vectors

vector = vectors.Vector(apikey='Your API key', referer='your-domain.com')
data = vector.tile('997', 256, 11, 327, 791)
with open('SF.png', 'w') as f:
    f.write(data)
